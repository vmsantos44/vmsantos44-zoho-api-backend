## Candidate Tier System

### Tier 1 (Highest Quality)
- **Requirements:**
  - 1+ years of Over-the-Phone Interpretation (OPI) experience
  - 4+ hours of formal training
  - Industry certifications preferred (CCHI, NBCMI, etc.)
- **Priority:** Fast-track for immediate placement
- **Typical Timeline:** 1-2 weeks from application to active

### Tier 2 (Mid-Level)
- **Requirements:**
  - 1+ years of on-site interpretation experience
  - Industry-specific training
  - No certification required
- **Priority:** Standard processing
- **Typical Timeline:** 2-4 weeks from application to active

### Tier 3 (Entry Level)
- **Requirements:**
  - Little to no formal experience
  - Minimal training
  - Bilingual with strong language skills
- **Priority:** Training required before placement
- **Typical Timeline:** 4-8 weeks from application to active

## Candidate Status Workflow

### Standard Pipeline
1. **New Candidate** ‚Üí Initial application received
2. **Application Received** ‚Üí Application under review
3. **Interview Scheduled** ‚Üí Interview set up with recruiter
4. **Documents Pending** ‚Üí Waiting for required documentation
5. **Onboarding** ‚Üí Active onboarding process
6. **Training** ‚Üí Undergoing company training
7. **Active Interpreter** ‚Üí Fully onboarded and working
8. **Not Qualified** ‚Üí Did not meet requirements

### Special Statuses
- **On Hold** ‚Üí Temporarily paused (candidate request or company decision)
- **Reapply Later** ‚Üí Not qualified now but could reapply in 6-12 months
- **Duplicate** ‚Üí Duplicate application (already in system)

## Key CRM Fields and Their Meanings

### Contact/Lead Fields
- **Full_Name** - Complete name of candidate
- **First_Name** - First/given name
- **Last_Name** - Last/family name
- **Email** - Primary email (always required)
- **Phone** - Primary contact number
- **Mobile** - Cell phone number
- **Status** - Current stage in pipeline
- **Tier** - Quality tier (1, 2, or 3)
- **Language** - Primary language(s) for interpretation
- **Location** - City, State, Country
- **Candidate_Recruitment_Owner** - Assigned recruiter
- **Created_Time** - When record was created
- **Modified_Time** - Last update timestamp
- **Tags** - Classification tags (Onshore, Offshore, Automated AI Review, etc.)

### Important Tags
- **Tier 1**, **Tier 2**, **Tier 3** - Quality tier
- **Onshore** - US-based
- **Offshore** - International
- **Spanish**, **Portuguese**, **French**, etc. - Language tags
- **Automated AI Review** - Processed by AI system
- **Documents Downloaded** - Resume/documents on file
- **No Resume** - No resume attached
- **Qualified** - Meets minimum requirements
- **Not Qualified** - Does not meet requirements

## Best Practices

### Email Communication
1. **Always personalize** - Use first name, reference their specific situation
2. **Be specific about next steps** - Clear actions, deadlines, and expectations
3. **Professional but friendly** - Balance professionalism with warmth
4. **Respect language preferences** - Use Portuguese/Spanish when appropriate
5. **Response time** - Aim for same-day response during business hours

### CRM Data Entry
1. **Always tag tier level** - Critical for prioritization
2. **Add location tag** - Onshore vs Offshore affects processing
3. **Note language tags** - Multiple languages possible
4. **Update status promptly** - Keeps pipeline accurate
5. **Add detailed notes** - Future recruiters need context

### Candidate Experience
1. **Set clear expectations** - Timeline, requirements, next steps
2. **Follow up regularly** - Don't let candidates go dark
3. **Be honest about timeline** - Better to under-promise, over-deliver
4. **Provide feedback** - Even for rejections, be constructive
5. **Maintain professionalism** - Every interaction reflects the company

## Troubleshooting Common Issues

### Candidate Not Responding
1. Check last contact date in notes
2. Try alternate phone number if available
3. Send follow-up email with clear deadline
4. If no response in 7 days ‚Üí Mark "On Hold" with note
5. If no response in 14 days ‚Üí Move to "Not Qualified" or "Reapply Later"

### Missing Documents
1. Check attachments in CRM
2. Review notes for previous requests
3. Send specific list of missing items
4. Provide multiple upload options
5. Set deadline (usually 48-72 hours)

### Duplicate Applications
1. Search both Contacts and Leads
2. Check email address (most reliable identifier)
3. Review creation dates
4. Merge duplicates (keep newest if data is equivalent)
5. Tag original as "Duplicate" and add note

### Wrong Language Assignment
1. Update language tags immediately
2. Reassign to appropriate language recruiter
3. Add note explaining correction
4. Notify candidate if timeline changes

### Recruiter Performance
- **Response time** - Target: < 24 hours for first contact
- **Time to interview** - Target: < 5 days for Tier 1, < 10 days for Tier 2/3
- **Onboarding completion** - Target: 80%+ completion rate
- **Time to active** - Target: 2 weeks (Tier 1), 4 weeks (Tier 2), 6 weeks (Tier 3)

### Pipeline Health
- **New candidates/week** - Track volume trends
- **Tier distribution** - Aim for 20% Tier 1, 50% Tier 2, 30% Tier 3
- **Language distribution** - Match to client demand
- **Onshore/Offshore ratio** - Balance based on needs
